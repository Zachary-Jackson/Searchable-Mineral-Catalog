This page is for help on how to search the Mineral Catalog.

{% endblock %} {% block content %}

To search the Mineral Catalog for any data that a mineral may contain (Search all data by Term) please use the box under the main Mineral Catalog logo. Just type in your query in one word and press enter.

The Search Alphabetically option allows you to choose the first starting letter for the minerals you want to search for.

The Search by Color option allows you to choose or search for a color of choice. If searching only use one word.

Search by Crystal System lets you search the Mineral Catalog via the seven kinds of Crystal Systems.

Seaching by Group allows you to choose a Group you want to filter the Mineral Catalog by.
